# Podcre Streetwear

Site oficial da Podcre Streetwear - marca de roupas urbanas com foco em estilo oversized, peças de qualidade e identidade visual forte.

## Sobre a Podcre

A Podcre é uma marca de streetwear que veste a quebrada, com peças pensadas para o dia a dia, noite e role, mantendo sempre a identidade e presenca de rua. Nossa coleção inclui:

- Camisetas oversized com grafismos pesados
- Conjuntos completos para uso diário
- Jaquetas e peças de outerwear
- Essenciais com base escura e shape forte

## Características do Site

- Navegação intuitiva por coleção, drops, fit guide e contato
- Integração direta com WhatsApp para pedidos e atendimento
- Visual moderno com foco em produtos e identidade de marca
- Responsivo para mobile e desktop

## Tecnologias Utilizadas

- React 19
- Vite
- Lucide React (ícones)
- CSS3 moderno

## Como Executar Localmente

```bash
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev

# Build para produção
npm run build

# Visualizar build localmente
npm run preview
```

## Deploy

O site está configurado para deploy fácil em plataformas como GitHub Pages, Netlify ou Vercel.

Para GitHub Pages:
```bash
npm run build
# Depois seguir o processo de deploy para gh-pages
```
